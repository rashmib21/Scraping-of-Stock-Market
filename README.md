# 📈 Scraping-of-Stock-Market

A Python-based project that scrapes and collects **historical stock market data** from **Angel One** and **Yahoo Finance**.

---

## 📁 Project Structure

```
Scraping-of-Stock-Market/
│
├── Angel One/          # Scripts to scrape historical data from Angel One
├── Yahoo Finance/      # Scripts to scrape historical data from Yahoo Finance
└── README.md
```

---

## 🚀 Features

- Scrape **historical stock data** from Angel One and Yahoo Finance
- Clean and process data using **Pandas**
- Structured and ready-to-analyze data output

---

## 🛠️ Technologies Used

| Tool | Purpose |
|------|---------|
| Python | Core programming language |
| Pandas | Data manipulation and analysis |

---

## ⚙️ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/rashmib21/Scraping-of-Stock-Market.git
   cd Scraping-of-Stock-Market
   ```

2. **Install dependencies**
   ```bash
   pip install pandas
   ```

3. **Run a scraper**
   ```bash
   # For Yahoo Finance
   cd "Yahoo Finance"
   python scraper.py

   # For Angel One
   cd "Angel One"
   python scraper.py
   ```

---

## 📊 Data Collected

Historical stock data including:

- Date
- Opening & Closing price
- High & Low price
- Volume
